# Mini Search Engine

C++ Mini Search Engine project using **OOP**, **BST**, and **Linked List**.

---

## Features
- Add multiple documents manually
- Search words across documents
- Track word frequency per document
- Print search results sorted by frequency

---

## How to Compile

```bash
g++ src/main.cpp -o MiniSearchEngine.exe
```

### On Linux / Mac:
./MiniSearchEngine.exe

### On Windows:
.\MiniSearchEngine.exe
